# Student Performance Prediction System

This project predicts student CGPA using Machine Learning.

## Technologies Used
- Python
- Pandas
- Scikit-Learn
- Flask
- HTML

## Models
- Linear Regression
- Random Forest
- XGBoost

## Features
- Study Hours
- Sleep Hours
- Attendance
- Coding Hours
- Social Media Usage
- Participation in Class

## Run Project

pip install -r requirements.txt

python train_model.py

python flash.py
